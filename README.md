# Venezia Invisibile 🎭

Un progetto web minimalista dedicato alla scoperta della **Venezia segreta**. Non la città dei grandi monumenti, ma quella degli angoli dimenticati, delle pietre che raccontano storie e degli scorci meno battuti.

## 🛠️ Tecnologie utilizzate
- **HTML5 / CSS3** (Layout responsive e minimalista)
- **GitHub Pages** (Hosting)
- **AI Assisted Design** (Per la ricostruzione storica delle guardie veneziane)

## 📍 Curiosità documentate
